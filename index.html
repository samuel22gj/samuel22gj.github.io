GPS Live

GPS Information Monitor Android App.

GPS Live is a GPS monitor app with Material Design. It can show any GPS information you want to know.

The following are some main features:

  • GPS provider state
  • Location information such as: Latitude, Longitude, Altitude, etc.
  • NMEA data (What's NMEA)
  • Detailed satellite information
  • Update frequency adjustment to save battery

MOD HD Plus

HD Android MOD System for Vehicle.

Based on customized Android system and floating design, MOD HO Plus is an Android App has rich multimedia player and intuitive UI. It can let each passenger to enjoy favorite entertainment.

The following are a part of main features:

  • Rich Multimedia (such as movies, music, games, internet, RSS news, etc.)
  • FM Radio
  • Live TV Streaming
  • Support Multi-language
  • Ordering System
  • Route Information
  • Driver Announcement
  • AD System
